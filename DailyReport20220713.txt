# 関数の高度な機能
## 変数スコープ、クロージャ
### コードブロック
・変数がコードブロック{...}の中で宣言された場合は、そのブロックの中でだけ見える
例：コードブロックの外から変数にアクセスする場合の処理結果
{
  // 外には見せる必要のない、ローカル変数で処理をする

  let message = "Hello"; // このコードブロックでだけ見える。

  alert(message); // Hello
}

 alert(message); // Error: message is not defined

if,for,whileなどの場合も同様

### ネストされた関数
・関数の内部で作成される関数をネストされた関数と呼ぶ
例：ネストされた関数getFullName()
function sayHiBye(firstName, lastName) {

  // 下で使うネストされたヘルパー関数です
  function getFullName() {
    return firstName + " " + lastName;
  }

  alert( "Hello, " + getFullName() );
  alert( "Bye, " + getFullName() );

}

また、新しいオブジェクトのプロパティまたはその自身の結果として、ネストされた関数を返すことができる
例:実行ごとに次の数を返すcounter関数
function makeCounter() {
  let count = 0;

  return function() {
    return count++;
  };
}

let counter = makeCounter();

alert( counter() ); // 0
alert( counter() ); // 1
alert( counter() ); // 2
